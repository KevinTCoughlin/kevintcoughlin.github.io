Solution: Proxy Issues With cURL

May 10th, 2013 | Comments

Quick Tip

Here’s a solution if you ever run into the following issue while running cURL:

$ [curl] 56: Received HTTP code 407 from proxy after CONNECT

and you believe that you have exhaustively removed all of your proxy environment variables i.e.:

  • http_proxy
  • https_proxy
  • all_proxy

Check if you created a profile for cURL ~/.urlrc, ikely in your home directory. You can find it by typing:

$ find / -type f -name ".curlrc"

I ran into this problem today when I forgot I had created the cURL profile months ago. In my home directory I had a .curlrc file with

cURL no longer went through the proxy once I removed the above line. Hopefully this will save at least one person from going insane like I did today. Hopefully I remember profile files from now on.
